# Software-Design-and-Architecture

## Description
This repository includes all **Diagrams** and **Java programs** that I developed during my **4th Semester** for the subject **"Software Design and Architecture"** in my **B.E Software Engineering Program**.

## Features
- **UML Diagrams** illustrating various software design patterns and architectures.
- **Java implementations** of design patterns such as Singleton, Factory, Observer, etc.
- Well-structured and documented code for better understanding.

## Prerequisites
- **Java Development Kit (JDK) 8 or higher**
- **Java-supported IDE** (Eclipse, IntelliJ IDEA, NetBeans) or a text editor with command-line support
- **Diagramming Tools** (Lucidchart, Microsoft Visio, or Draw.io) for viewing and editing diagrams

## How to Run
Clone the repository:
   ```sh
   git clone https://github.com/yourusername/Software-Design-and-Architecture.git
   ```
